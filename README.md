# credit-default-analysis
This project analyzes credit card default risk using a dataset of about 30,000 clients. The goal was to identify key predictors of default and evaluate predictive models using multiple statistical techniques.

Methods Used:
* Logistic Regression (Stepwise Selection)
* Principal Component Analysis (PCA)
* Canonical Correlation Analysis (CCA)
* Exploratory Factor Analysis (EFA)

Results:
* Model achieved ~76% accuracy and AUC ≈ 0.73
* Strongest predictor: payment delays (ANY_DELAY)
* Better at predicting non-default vs default clients

Insights:
* Payment behavior is more predictive than demographics
* Higher credit limits leads to lower default risk
* Late payments strongly increase default probability
